## API Reference

### `crystallize(domainData, options)`

Crystallizes knowledge domain into crystal structure.

**Parameters:**

- `domainData` (Object/Array): Source knowledge data
- `options` (Object): Crystallization options
  - `depth` (Number): Crystallization depth (1-10)
  - `connections` (String): 'minimal', 'standard', 'maximum'
  - `compression` (String): 'none', 'low', 'medium', 'high', 'maximum'

**Returns:** Crystal object

**Example:**

```javascript
const crystal = await omniscientHDR.crystallize(quantumPhysicsData, {
  depth: 8,
  connections: "maximum",
  compression: "high",
});
```

### `accessKnowledge(crystalId, query)`

Accesses knowledge from crystal structure.

**Parameters:**

- `crystalId` (String): Crystal identifier
- `query` (Object): Knowledge query

**Returns:** Knowledge results

**Example:**

```javascript
const knowledge = await omniscientHDR.accessKnowledge(crystal.id, {
  concept: "quantum-entanglement",
  depth: "expert",
  format: "practical",
});
```

### `infuseKnowledge(aiModel, crystalId)`

Infuses crystallized knowledge into AI model.

**Parameters:**

- `aiModel` (Object): Target AI model
- `crystalId` (String): Crystal identifier

**Returns:** Infusion result

**Example:**

```javascript
const result = await omniscientHDR.infuseKnowledge(aiModel, crystal.id);
console.log(`Knowledge infused: ${result.conceptsTransferred} concepts`);
console.log(`Acquisition time: ${result.duration}ms (100x faster)`);
```
