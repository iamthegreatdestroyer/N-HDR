## Troubleshooting

### Issue: Swarm deployment fails

**Solution:** Check that all dependencies are installed and node version is 18+

### Issue: Monitoring dashboard not displaying

**Solution:** Ensure terminal supports Unicode characters and has sufficient width (80+ columns)

### Issue: Timestamp verification fails

**Solution:** Verify write permissions for `timestamps/` directory

### Issue: Out of memory errors

**Solution:** Reduce swarm bot count or increase Node.js heap size:

```bash
node --max-old-space-size=4096 src/cli/execute-hdr-protocol.js
```
